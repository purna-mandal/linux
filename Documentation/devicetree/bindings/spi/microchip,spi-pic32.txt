Microchip PIC32 SPI device

Required properties:
- compatible : should be "microchip,pic32-spi".
- reg: Address and length of the register set for the device
- interrupts: Should contain all three spi interrupts in sequence
              of <fault-irq>, <receive-irq>, <transmit-irq>.
- cs-gpios:  Specifies the gpio pins to be used for chipselects.
- clocks: phandles to input clocks. Maximum two possible clocks
	  can be provided (pbxclk, refoxclk).
- microchip,clock-indices: specify, 1 if clocks provides refclk phandle.
- pinctrl: specify correct pinctrl name & pinctrl group.

Example:

spi1:spi@0xbf821000 {
        compatible = "microchip,pic32-spi";
        reg = <0xbf821000 0x200>;
        interrupt-parent = <&evic>;
        interrupts = <SPI1_FAULT DEFAULT_INT_PRI IRQ_TYPE_NONE>,
                     <SPI1_RECEIVE_DONE DEFAULT_INT_PRI IRQ_TYPE_NONE>,
                     <SPI1_TRANSFER_DONE DEFAULT_INT_PRI IRQ_TYPE_NONE>;
        clocks = <&PBCLK2>;
#if 0
        clocks = <&REFCLKO1>;
        microchip,clock-indices = <1>;
#endif
        pinctrl-names = "default";
        pinctrl-0 = <&pinctrl_spi1>;
        cs-gpios = <&pioD 4 0>;
        flash: m25p80@0 {
		#address-cells = <1>;
                #size-cells = <0>;
                compatible = "m25p80";
                spi-max-frequency = <25000000>;
                reg = <0>;
	};
};
